Important git commands:
1. create txt file in working directory: echo "content_of_file" >> file_name.txt
2. view contents of txt file with name file_name: cat file_name.txt
3. adding txt file to staging area: git add file_name.txt
4. moving contents from staging area to local repository: git commit -m "Process explanation"
5. moving contents from local repository to github repository: git push origin branch_name
6. checking status of local area wrt remote area: git status
